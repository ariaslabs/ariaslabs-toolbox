## Ariaslabs Toolbox  
___

__Install__

```javascript
npm i ariaslabs-toolbox
```
____
__log()__
log is a quick way to log anything out while keeping your code clean

```javascript
log("Hello, World")

//Output: Hello, World
```
__sleep()__
Sleep is a delay function to delay logic. 

```javascript
async function myFunction() {
  await sleep(1000) //sleeps for 1 second
  //do stuff
}
```
